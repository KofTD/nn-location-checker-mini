# Процесс работы 

Для начала работы необходимо иметь ruff >= 0.15.9

Также установите зависимости для разработки:
```bash
(.venv) $ pip install --group dev
```

## Соглашение о commit'ах

- feat: — новая функциональность
- fix: — исправление бага
- docs: — изменения в документации
- test: — добавление / правка тестов
- style: — форматирование, не влияющее на логику
- refactor: — рефакторинг кода
- perf: — улучшение производительности
- ci: — изменения в CI/CD скриптах
- chore: — обслуживание (обновление зависимостей, настройки)'

Примеры

- `feat: add dark mode`
- `refactor: simplify input logic`
